# Gina 学英语 — 产品方案

> **约束条件**
> - 团队：Mr王（组织/架构）+ Miss白（配合），无外部人员
> - 配图用 AI 生成，免费方案够用
> - 无办学许可/教育类资质 → 产品定位走"工具"路线，避开教育资质要求
> - 目标用户：上海小学生，教材：**沪教版二年级下册**
> - 上线目标：**今晚上线**
> - 域名：免费域名（Cloudflare Pages / Vercel 免费域名）

## 一、产品形态选择

### 推荐：**网页版（PWA）优先，后续转微信小程序**

**MVP 阶段只做网页版**，原因：
- 3 人团队资源有限，先做一套代码
- PWA 安装后可像原生 App 一样使用
- 后续用 Taro/uni-app 快速转小程序
- 家长端管理方便，无需安装即可访问

## 二、目标年级和词库范围

### 核心目标：上海小学 3-5 年级

| 年级 | 词汇量 | 教材版本 |
|------|--------|----------|
| 一年级 | ~50 词 | 牛津上海版 |
| 二年级 | ~80 词 | 牛津上海版 |
| 三年级 | ~100 词 | 牛津上海版 |
| **四年级** | **~120 词** | **牛津上海版** |
| **五年级** | **~150 词** | **牛津上海版** |
| 六年级 | ~180 词 | 牛津上海版 |

### 词库策略
1. **MVP 阶段**：覆盖 3-5 年级牛津上海版核心词汇（约 300-400 词）
2. **词库结构**：每个单词包含
   - 英文单词 + 音标
   - 中文释义（可多义项）
   - 例句
   - 英音 + 美音发音（使用 Web Speech API 或 TTS 服务）
   - AI 配图（每个单词配一张图，帮助记忆）
3. **数据来源**：
   - 手动整理牛津上海版教材词表（初期可靠）
   - 后续接入开源词库 API（如 ECDICT、WordNet）
   - 支持用户自定义词库上传
4. **AI 配图方案**：
   - MVP 阶段用免费 AI 工具（如 HuggingFace Spaces、Stable Diffusion WebUI 本地部署）
   - 每个单词生成 1 张简洁插画（扁平风格，适合儿童）
   - 配图缓存到 CDN，减少重复生成

## 三、核心功能设计

### 3.1 背单词模块
- **卡片模式**：正面显示英文单词，点击翻转看中文释义 + 例句
- **拼写模式**：听音拼词 / 看中文输入英文
- **跟读模式**：点击播放发音，学生跟读（可选录音对比）
- **每日目标**：可设置每日新词数量（建议 5-15 个/天）

### 3.2 复习机制（核心差异化）
- **艾宾浩斯遗忘曲线**：系统自动安排复习时间
  - 第 1 天学习 → 第 2 天复习 → 第 4 天复习 → 第 7 天 → 第 15 天 → 第 30 天
- **智能算法**：根据学生答题正确率动态调整复习间隔
  - 答错 → 缩短间隔（次日再复习）
  - 连续答对 → 延长间隔
- **错词本**：自动收集答错的单词，单独复习

### 3.3 游戏化闯关
- **每日闯关**：完成当日学习 + 复习任务解锁关卡
- **单词消消乐**：连线配对游戏（英文-中文配对）
- **拼写竞速**：限时拼写，看谁最快最准
- **积分排行榜**：按班级/好友排行（可选）
- **成就系统**：连续打卡、词汇量里程碑等

### 3.4 家长端
- **学习报告**：每日/每周学习时长、完成度、正确率
- **进度追踪**：查看孩子掌握哪些单词、哪些需要加强
- **设置管理**：调整每日学习目标、暂停学习

### 3.5 辅助功能
- **离线使用**：PWA 缓存支持离线背词
- **深色模式**：保护视力
- **家长锁**：防止孩子随意修改设置

## 四、合规与定位

### 产品定位
- **定位**：学习工具（非教育机构、非培训课程）
- **宣传口径**："单词学习工具" / "背单词 App"，避免"培训""提分""课程"等敏感词
- **用户协议**：注明"仅供学习辅助，不构成教学建议"

### 资质规避
- 不做真人教师授课、不做直播课、不做作业批改
- 纯工具属性：背单词 + 复习 + 游戏化
- 不涉及收费培训，免费使用（后期可考虑增值功能）

## 五、技术选型

### 前端
| 技术 | 说明 |
|------|------|
| **React + TypeScript** | 主框架，类型安全 |
| **Vite** | 构建工具，开发体验好 |
| **Tailwind CSS** | 样式方案，快速开发 |
| **PWA (workbox)** | 离线支持、安装到桌面 |
| **React Router** | 路由管理 |
| **Zustand** | 状态管理（轻量） |

### 后端
| 技术 | 说明 |
|------|------|
| **FastAPI (Python)** | API 服务，开发快、性能好 |
| **SQLite** | 初期数据库（数据量小时足够） |
| **Redis** | 缓存、排行榜（可选） |
| **Web Speech API** | 前端发音（MVP 阶段免后端 TTS） |

### 部署
| 方案 | 说明 |
|------|------|
| **Vercel / Netlify** | 前端托管，免费额度足够 |
| **Railway / Fly.io** | 后端部署，免费额度起步 |
| **Cloudflare Pages** | 免费 CDN + PWA 分发 |

### 总成本
- **MVP 阶段**：几乎零成本（全用免费额度）
- **用户量增长后**：约 ¥100-300/月
