## 四、MVP 核心功能（今晚上线版）

| 优先级 | 功能 | 说明 |
|---|---|---|
| P0 | 卡片背词 + 发音 | 核心功能 |
| P0 | SM-2 复习算法 | 根据记忆程度自动安排复习 |
| P0 | 错词本 | 自动收集，重点复习 |

> 闯关、打卡、配图、家长端全部砍掉，今晚只上核心功能。
